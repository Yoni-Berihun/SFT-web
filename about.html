Hawassa-born project

Built for students, shaped by educators

EduFinance is an open learning lab that turns budgeting lessons into immersive, data-informed simulations powered by clean, auditable code.

Join community call Contact us

Mission

Demystify finance for every cohort

We translate complex budgeting flows into transparent UI so facilitators can walk students through each decision, slider, and chart with confidence.

Vision

Curriculum-ready building blocks

EduFinance aims to be the reference implementation for modern, inclusive financial literacy labs—ready to fork, remix, and ship across campuses.

Pillars

What makes EduFinance different?

Each release focuses on clarity, accountability, and collaborative learning so instructors spend time teaching, not debugging.

Learning-first flows

Every screen narrates its own purpose with inline helper copy, status chips, and meaningful defaults.

Visual explainers

Doughnut charts, bar comparisons, and live breakdowns keep students oriented when numbers shift.

Community governance

Faculty partners and students submit pull requests that shape accessibility, language, and features.
